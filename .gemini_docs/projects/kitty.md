# Kitty Configuration

Primary terminal emulator configuration for Kitty.

## Aesthetic & Performance
- **Font:** Iosevka Custom (12pt).
- **Transparency:** 0.75 opacity with dynamic background opacity enabled.
- **Colors:** OLED-friendly (Pure black background `#000000`).
- **Layouts:** `splits` layout enabled.

## Key Files
- `kitty.conf`: Main configuration.
- `mappings.conf`: Custom keybindings.
- `tabs.conf`: Tab bar styling.
- `kitty-themes/`: Collection of color schemes (Symfonic.conf is default).
