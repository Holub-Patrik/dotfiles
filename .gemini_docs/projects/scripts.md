# Scripts & Tools

A collection of personal utility scripts for development and system management.

> [!WARNING]
> This directory (`~/.local/bin`) is shared with other applications. **DO NOT** edit or modify scripts that are not listed below, as they are likely system or application-managed binaries.

## Tracked Scripts
- `cmake-build`: A CMake wrapper for C++ projects that automates setup and compilation.
- `forgor`: (WIP) A notification tool to alert about uncommitted or unpulled changes in Git repositories.
- `remove_spaces`: A legacy script for batch renaming files to remove spaces (kept for reference).

## Hyprland Helper Scripts
These are located in `hypr/scripts/`:
- `brightness.sh`: Monitor brightness control helper.
- `calculator.sh`: A small floating terminal running `qalc`.
- `powermenu.sh`: System power management menu.
- `screenshot.sh`: Screenshot utility.
- `volume.sh`: Volume control helper.
- `wallpaper.sh`: Wallpaper switcher.
