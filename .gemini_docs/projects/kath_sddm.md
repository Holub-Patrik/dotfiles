# SDDM Theme (kath_sddm)

A custom SDDM login theme designed for the PidgyDots setup.

## Features
- **QML Based:** Modern and customizable interface.
- **Assets:** Custom SVGs for system actions (Shutdown, Reboot, etc.).
- **Typography:** Uses `pixelon.regular.ttf`.
- **Background:** Supports both image and video backgrounds (mp4).

## Structure
- `Main.qml`: Main theme logic.
- `Components/`: Reusable QML components for login form, clock, etc.
- `hyprland_kath.conf`: Associated Hyprland config for the SDDM session.
