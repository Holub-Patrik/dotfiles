# Zsh Configuration

Zsh shell configuration focused on productivity and developer tools.

## Key Features
- **Prompt:** Powerlevel10k with instant prompt enabled.
- **Completion:** Advanced completion settings (`zstyle`).
- **Integration:** SSH agent management, path configuration for Odin/OLS.
- **Keybindings:** Emacs-style (`bindkey -e`), custom clear screen behavior.
